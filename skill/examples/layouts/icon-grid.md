# 서비스 소개

## 핵심 기능 4가지

- 🚀 **빠른 배포** — 원클릭으로 프로덕션 배포
- 🔒 **보안 강화** — 자동 취약점 스캔 및 패치
- 📊 **실시간 모니터링** — 대시보드로 상태 확인
- 🔄 **자동 롤백** — 장애 감지 시 즉시 복구

> Note: 이모지 + 볼드 키워드 + 설명 패턴. 이모지가 있으면 리스트 아이콘 힌트로 유지됨.

---

## 팀 역할

- 👨‍💻 **프론트엔드** — React, 사용자 경험 담당
- 🛠️ **백엔드** — Spring Boot, API 설계
- 🗄️ **데이터** — 파이프라인 구축, 분석
- ☁️ **인프라** — AWS, K8s, CI/CD

> Note: 팀 구성이나 역할 분담을 그리드처럼 보여줄 때 적합.

---

## 왜 우리를 선택해야 하는가

- ✅ **검증된 기술** — 3년간 무장애 운영
- 💰 **합리적 가격** — 경쟁사 대비 40% 저렴
- 🤝 **전담 지원** — 24시간 기술 지원
- 🌏 **글로벌 인프라** — 5개국 리전 운영

> Note: 영업/제안 발표에서 자주 쓰이는 USP(Unique Selling Point) 나열 패턴.
