# 비교 분석

## A안 vs B안

| 항목 | A안 (자체 개발) | B안 (외부 솔루션) |
|------|----------------|-----------------|
| 비용 | 높음 (인건비) | 중간 (라이선스) |
| 기간 | 6개월 | 2개월 |
| 커스터마이징 | 자유로움 | 제한적 |
| 유지보수 | 내부 부담 | 벤더 지원 |

> Note: 테이블 기반 비교. 의사결정 슬라이드에서 가장 많이 쓰이는 패턴.

---

## Before vs After

**Before:** 수동 배포, 평균 **2시간** 소요

**After:** CI/CD 파이프라인, 평균 **5분** 소요

> 95% 시간 단축 달성

> Note: stat-card + callout 조합으로 전후 비교 강조. 숫자가 있으면 더 효과적.

---

## 기술 스택 비교

**현재 스택:**
- Monolith (Spring Boot)
- 단일 DB (MySQL)
- 수동 배포

**목표 스택:**
- MSA (Spring Cloud + K8s)
- DB 분리 (MySQL + Redis + MongoDB)
- GitOps 자동 배포

> Note: 두 리스트를 나란히 배치하는 텍스트 비교 패턴.
